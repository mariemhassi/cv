CV en Ligne - Mariem Hassi
Description

Ce projet est un CV en ligne interactif pour Mariem Hassi, étudiante en 2ᵉ année en Informatique Appliquée.
Le CV est conçu avec HTML, CSS et JavaScript, et utilise Bootstrap pour la mise en page responsive.

Le design se compose de deux colonnes :

Sidebar : photo de profil, contact, langues et loisirs

Contenu principal : formations, expériences et compétences

Le CV intègre des animations CSS :

Rotation de la photo au survol

Zoom du nom de l’étudiant au survol

Et des fonctionnalités JavaScript :

Cliquer sur les titres de sections (Formations, Expériences, Compétences) permet de masquer ou afficher le contenu correspondant.

Il est responsive et s’adapte aux écrans d’ordinateur, tablette et mobile.

Technologies utilisées

HTML5 : structure du contenu

CSS3 : mise en forme et animations

JavaScript : interactivité (toggle du contenu des sections)

Bootstrap 5 : mise en page responsive

Google Fonts : Playfair Display et Poppins
